# Roadmap

## Product Goal

転職者との面談内容をもとに、その内容にマッチする企業を外部企業データベースから選出する。

このプロダクトは企業を自動決定するものではなく、面談内容と企業情報を照合し、条件や志向に合う企業を選出するための補助ツールとする。

## Selection Style

点数づけは使わない。選出結果は、以下の判断区分で表現する。

- マッチ度が高い
- マッチしている
- 条件確認が必要
- 現時点ではマッチが弱い

各企業には以下を必ず含める。

- 選出区分
- 選出理由
- 懸念点
- 追加確認すべき質問

## Phase 1: Text-Based MVP

目的: 音声や外部DB連携の前に、面談テキストからマッチする企業を選出できるか検証する。

作るもの:

- 面談テキスト入力
- 面談要約
- 転職者プロフィール整理
- サンプル企業DB
- 選出区分つきの企業候補出力

完了条件:

- 1つの面談テキストから、転職者プロフィールJSONが生成される
- サンプル企業DBから候補企業が選出される
- 選出理由、懸念点、追加質問が出力される
- 点数づけを使わない

## Phase 2: Matching Logic Improvement

目的: 面談内容と企業情報の照合品質を上げる。

作るもの:

- 必須条件による除外ロジック
- NG条件の検出
- 一致点と懸念点の分離
- 選出区分の判定ルール
- 選出結果の説明テンプレート

## Phase 3: External Company Database Integration

目的: サンプル企業DBから、実際に利用可能な企業データへ接続する。

作るもの:

- 企業データ入力層
- 企業データの正規化
- 求人・企業情報の取得
- 障害時のエラーハンドリング
- 環境変数管理

## Phase 4: Advisor UI

目的: キャリアアドバイザーが面談後に、マッチする企業を確認しやすくする。

作るもの:

- 面談テキスト入力画面
- 要約表示
- 転職者プロフィール表示
- 選出企業一覧
- 選出理由・懸念点・追加質問の表示
- 選出区分での絞り込み
